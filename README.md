### Quick Install Commands Summary

To add components from these registries directly to your workspace, run the respective CLI command:

- **Animate UI**: `npx shadcn@latest add <component-name>` (local registry)
- **Tool UI**: `npx shadcn@latest add https://www.tool-ui.com/r/<component-name>.json` (local registry)
- **BoldKit**: `npx shadcn@latest add https://boldkit.dev/r/<component-name>.json` (local registry)
- **Chamaac UI**: `npx shadcn@latest add https://www.chamaac.com/r/<component-name>.json` (local registry)
- **Lumen UI**: `npx shadcn@latest add https://ui.unlumen.com/r/<component-name>.json` (free components)
